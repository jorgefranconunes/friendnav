= Third-party libraries used by FriendNav





* http://getbootstrap.com/[Bootstrap] -- HTML, CSS, and JS framework.

* https://github.com/JAAulde/cookies[cookies] -- Javascript library
  for accessing and manipulating HTTP cookies in the web browser.

* https://d3js.org/[D3.js] -- JavaScript library for manipulating
  documents based on data.

* http://documentup.com/kriskowal/q/[Q] -- Javascript library
  supporting promises.
